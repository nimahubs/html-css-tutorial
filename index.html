Introduction to HTML

10 min read • Beginner

What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page using a series of elements that tell the browser how to display content.

💡 Did you know? HTML was created by Tim Berners-Lee in 1991. The latest version is HTML5.

Your First HTML Document

Every HTML document follows a basic structure. Let's look at a simple example:

index.html 
<!-- This is a comment -->
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Page</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is my first web page.</p>
</body>
</html>
▶ $ Output: Hello, World! | This is my first web page.

Breaking Down the Structure

  • <!DOCTYPE html> - Tells the browser this is an HTML5 document
  • <html> - The root element of the page
  • <head> - Contains meta information about the document
  • <body> - Contains the visible content of the page

Create a new HTML file and add the basic structure. Try changing the title and adding a paragraph.

Introduction to CSS

12 min read • Beginner

What is CSS?

CSS (Cascading Style Sheets) is the language used to style HTML elements. It controls the visual presentation of your web pages, including colors, layouts, fonts, and animations.

Three Ways to Add CSS

1. Inline CSS

Inline Style 
<p style="color: blue; font-size: 18px;">
    This text is blue and 18px.
</p>

2. Internal CSS

Internal Style 
<head>
    <style>
        p {
            color: blue;
            font-size: 18px;
        }
    </style>
</head>

3. External CSS (Recommended)

styles.css 
/* styles.css */
p {
    color: blue;
    font-size: 18px;
    line-height: 1.6;
}
▶ $ Link: <link rel="stylesheet" href="styles.css">
💡 Best Practice: Always use external CSS files for better organization and maintainability.

HTML Elements

15 min read • Beginner

What are HTML Elements?

HTML elements are the building blocks of HTML pages. An element typically consists of an opening tag, content, and a closing tag.

Element Structure 
<!-- Element structure -->
<tagname>Content goes here</tagname>

<!-- Example -->
<p>This is a paragraph element.</p>
<h1>This is a heading element.</h1>

Block vs Inline Elements

Block Elements

  • <div>
  • <p>
  • <h1> to <h6>
  • <ul>, <ol>
  • <li>
  • <table>
  • <form>

Start on a new line, take full width

Inline Elements

  • <span>
  • <a>
  • <strong>
  • <em>
  • <img>
  • <button>

Flow within text, only take needed width

examples.html 
<!-- Block element - takes full width -->
<div style="background: lightblue;">
    This div takes full width
</div>

<!-- Inline elements - flow in line -->
<p>
    This is <strong>bold</strong> and 
    <em>italic</em> text.
</p>

HTML Attributes

12 min read • Beginner

What are HTML Attributes?

Attributes provide additional information about HTML elements. They are always placed in the opening tag and come in name/value pairs.

attributes.html 
<!-- href attribute for links -->
<a href="https://google.com">Visit Google</a>

<!-- src and alt attributes for images -->
<img src="photo.jpg" alt="My Photo" width="300">

<!-- id and class attributes -->
<div id="header" class="container">
    Content here
</div>
▶ $ Attributes customize element behavior and appearance

Common Attributes

Attribute Used With Purpose
href <a> Link destination
src <img>, <script> Source file path
alt <img> Alternative text
class Any element CSS class name
id Any element Unique identifier

Links & Images

12 min read • Beginner

Creating Links

Links are created using the <a> (anchor) tag with the href attribute.

links.html 
<!-- External link -->
<a href="https://google.com">Visit Google</a>

<!-- Link that opens in new tab -->
<a href="https://google.com" target="_blank">
    Open in new tab
</a>

<!-- Internal link -->
<a href="about.html">About Us</a>

<!-- Link to section on same page -->
<a href="#contact">Contact Section</a>

Adding Images

images.html 
<!-- Basic image -->
<img src="photo.jpg" alt="Description">

<!-- Image with size -->
<img src="logo.png" 
     alt="Logo"
     width="200"
     height="100">

<!-- Image as link -->
<a href="https://google.com">
    <img src="logo.png" alt="Google">
</a>
▶ $ Always include alt text for accessibility!
⚠️ Important: Always add the alt attribute to images. It's used by screen readers and displayed when images fail to load.

Understanding the Box Model

Every HTML element is a box consisting of: content, padding, border, and margin.

Margin
Border
Padding
Content
box-model.css 
/* Box Model Properties */
.box {
    /* Content */
    width: 300px;
    height: 200px;
    
    /* Padding (inside border) */
    padding: 20px;
    /* Or individual sides */
    padding-top: 10px;
    padding-right: 15px;
    
    /* Border */
    border: 2px solid #333;
    
    /* Margin (outside border) */
    margin: 20px auto;
}

/* Box-sizing: Include padding in width */
* {
    box-sizing: border-box;
}
▶ $ Total width = width + padding + border + margin

Margin & Padding

10 min read • Beginner

Spacing Properties

spacing.css 
/* Margin - Space OUTSIDE */
.box {
    /* All sides */
    margin: 20px;
    
    /* Vertical | Horizontal */
    margin: 10px 20px;
    
    /* Top | Right | Bottom | Left */
    margin: 10px 20px 15px 5px;
    
    /* Center horizontally */
    margin: 0 auto;
}

/* Padding - Space INSIDE */
.card {
    padding: 20px;
    padding-left: 30px;
}

/* Margin collapsing */
.element {
    margin-bottom: 20px;
}
.element + .element {
    margin-top: 20px;
}
/* Result: 20px (not 40px) */
▶ $ Margin collapses vertically; padding never collapses

Flexbox Container

flexbox.css 
/* Flex Container */
.container {
    display: flex;
    
    /* Direction */
    flex-direction: row; /* | column | row-reverse */
    
    /* Wrapping */
    flex-wrap: wrap;
    
    /* Alignment */

    align-items: center;
    gap: 20px;
}

/* Flex Items */
.item {
    flex: 1; /* Equal widths */
}

.item-wide {
    flex: 2; /* Takes 2x space */
}

Responsive Design

18 min read • Intermediate

Media Queries

responsive.css 
/* Mobile First Approach */
.container {
    width: 100%;
    padding: 10px;
}

/* Tablet */
@media (min-width: 768px) {
    .container {
        max-width: 720px;
        margin: 0 auto;
    }
}

/* Desktop */
@media (min-width: 1024px) {
    .container {
        max-width: 960px;
    }
}

/* Responsive Units */
.title {
    font-size: clamp(1rem, 3vw, 2rem);
}

.box {
    width: min(100%, 500px);
    height: min(100vh, 600px);
}
▶ $ Common breakpoints: 576px, 768px, 992px, 1200px
